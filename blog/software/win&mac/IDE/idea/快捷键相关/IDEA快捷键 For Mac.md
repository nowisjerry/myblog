## Mac键盘符号和修饰键说明

下面有些功能，Mac小键盘空间不足，没有单键实现，转而使用组合键来实现。

- `⎋` Escape (Esc)
- `⌘` Command 命令 = Windows的Windows键
- `⇧` Shift 切换
- `⌥` Option 选项 = Windows 的Alt键
- `⌃` Control 控制
- `↩︎` Return/Enter 换行
- `⌫` Delete 向左删除
- `⌦` 向右删除 = Fn + Delete
- `⇞` Page Up = `Fn + ↑`
- `⇟` Page Down = `Fn + ↓`
- `Home` = `Fn + ←左方向键`
- `End` = `Fn + →右方向键`
- `⇥` 右制表符（Tab键）
- `⇤` 左制表符 = `Shift + Tab`
